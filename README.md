# L3TT3R

Welcome to **L3TT3R**, a fast-paced, competitive word game where quick thinking is your best weapon!

## 🎮 The Game

L3TT3R pits you against another player in a head-to-head race to find the perfect word. It's simple to learn but delightfully challenging under pressure.

Here is how a round works:
1. **Pick the Letters**: One player chooses the starting letter, and the other chooses the ending letter.
2. **The Race Begins**: A 99-second countdown starts. Both players scramble to type a valid English word that begins and ends with those exact letters and meets the host's minimum length requirement.
3. **First to Guess Wins**: The first player to submit a valid word wins the round and earns a point! 
4. **Learn as You Play**: Every winning word is instantly translated into Chinese, giving you a fun way to pick up some new vocabulary while you compete.

## 🏆 How to Play

- **Host a Match**: Set your own rules! Choose the target score needed to win the game and the minimum character length for valid words.
- **Join a Match**: Jump straight into the action and challenge an opponent. 
- **Can't think of a word?**: You can choose to "Pass" the round, but both players must pass to draw the round!

Think fast, type faster, and see if you have what it takes to become the L3TT3R champion!
